📋 如何使用：

  1. 点击上方按钮下载每个尺寸的图标
  2. 将下载的图标重命名为 icon-192.png、icon-512.png、icon-1024.png
  3. 将图标放在项目根目录下（和 index.html 同一文件夹
  4. 访问您的游戏现在就是完整的 PWA 应用了！

🚀 安装 PWA 应用：

  1. 在 Chrome/Edge 浏览器中打开 index.html
  2. 确保使用 HTTPS 或 localhost
  3. 点击地址栏右侧的「安装应用」按钮
  4. 完成！游戏现在会像桌面应用一样运行！
