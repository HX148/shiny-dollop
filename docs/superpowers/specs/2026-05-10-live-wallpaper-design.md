# 手机动态壁纸应用设计方案

## 1. 项目概述

**项目名称**: LiveWallpaper
**项目类型**: Android原生应用
**核心功能**: 从本地选择视频，编辑后设为动态壁纸
**目标用户**: 希望个性化手机桌面的Android用户

## 2. 技术架构

### 技术栈
- **语言**: Kotlin 1.9+
- **UI框架**: Jetpack Compose + Material Design 3
- **视频播放**: ExoPlayer (Media3)
- **视频处理**: FFmpeg (via FFmpegKit) - 视频裁剪
- **架构模式**: MVVM + Clean Architecture
- **依赖注入**: Hilt
- **状态管理**: StateFlow + Compose State
- **动态壁纸**: Android Live Wallpaper Service API

### 模块划分
```
app/
├── data/                    # 数据层
│   ├── repository/          # 仓库实现
│   └── video/              # 视频处理
├── domain/                  # 领域层
│   ├── model/              # 数据模型
│   └── repository/          # 仓库接口
├── presentation/            # 表现层
│   ├── ui/                 # Compose UI
│   ├── viewmodel/          # ViewModel
│   └── theme/              # 主题配置
└── service/                 # Android服务
    └── wallpaper/          # 动态壁纸服务
```

## 3. UI/UX 设计

### 页面结构
1. **首页** - 视频库浏览（瀑布流布局）
2. **视频预览页** - 全屏预览和播放控制
3. **编辑页** - 裁剪和时间范围选择
4. **设置页** - 壁纸服务配置

### 导航流程
```
首页 → 选择视频 → 预览 → 编辑 → 设为壁纸
  ↓
设置页（配置壁纸服务）
```

### 动效设计
- **页面转场**: 共享元素动画 + 淡入淡出
- **列表加载**: 卡片依次飞入动画
- **按钮交互**: 波纹反馈 + 缩放动画
- **视频预览**: 手势滑动快进/快退
- **编辑界面**: 裁剪框支持拖拽和缩放

### 视觉风格
- **主题**: 深色主题为主，护眼且突出视频内容
- **配色**: 紫蓝色渐变主色调 (#6366F1 → #8B5CF6)
- **圆角**: 16dp卡片圆角
- **阴影**: 柔和阴影增加层次感

## 4. 核心功能

### 4.1 视频选择
- 系统相册选择器
- 最近使用的视频快速访问
- 视频缩略图预览

### 4.2 视频预览
- 全屏播放预览
- 播放/暂停控制
- 进度条拖动
- 静音开关

### 4.3 视频编辑
- **裁剪**: 拖拽框选择视频画面区域
- **时间范围**: 滑块选择起始和结束时间
- **实时预览**: 编辑效果即时显示

### 4.4 设为壁纸
- 使用Android Live Wallpaper Service
- 后台保存视频文件
- 一键应用到桌面
- 锁屏壁纸选项

### 4.5 壁纸管理
- 已设置壁纸列表
- 快速切换不同壁纸
- 删除壁纸

## 5. 数据流

### 视频设置壁纸流程
```
1. 用户选择视频 → ContentResolver获取URI
2. 进入预览页 → ExoPlayer播放预览
3. 进入编辑页 → 用户裁剪/选择时间
4. 点击设为壁纸 →
   a. FFmpeg处理视频(裁剪+时间范围)
   b. 保存处理后视频到应用私有目录
   c. 启动LiveWallpaperService
   d. 应用新壁纸
```

### 状态管理
- **ViewModel**: 管理UI状态和数据
- **StateFlow**: 响应式状态更新
- **SavedStateHandle**: 配置变更时保存状态

## 6. 权限需求
- `READ_EXTERNAL_STORAGE` - 读取视频文件
- `READ_MEDIA_VIDEO` (Android 13+) - 读取视频
- `SET_WALLPAPER` - 设置壁纸

## 7. 错误处理
- 无视频权限: 引导用户去设置页面授权
- 视频格式不支持: 显示友好提示
- 视频文件损坏: 跳过并提示
- 壁纸设置失败: 重试机制 + 错误日志

## 8. 性能优化
- 视频缩略图缓存
- 懒加载视频列表
- 后台处理视频避免阻塞UI
- 壁纸服务内存优化
