# 🎮 机甲对决 - 终极版

一款精彩的本地和网络对战游戏，支持多种主题风格和两位玩家！

---

## ✨ 游戏特色

- 🎯 **本地双人对战** - 在一台电脑上玩
- 🌐 **网络对战** - 和朋友在线对战
- 🎭 **8位英雄** - 蓝骑士、红战士、绿忍者、紫法师等
- 🎨 **5种主题** - 科技、日式、西部、赛博朋克、未来科幻
- 📱 **PWA 支持** - 像应用一样安装使用
- 💻 **桌面应用** - Electron 版本
- 📊 **完整进度** - 英雄选择、禁用、战斗
- ⚔️ **技能系统** - 每个英雄都有独特技能和奥义

---

## 🚀 快速开始

### 方式一：PWA 版本（推荐！）

无需安装，直接浏览器运行！

```bash
cd src
python3 -m http.server 8080
# 然后访问 http://localhost:8080/index.html
```

在 Chrome/Edge 浏览器中点击地址栏的「安装」按钮，像桌面应用一样运行！

详细说明见 [docs/PWA.md](docs/PWA.md)

### 方式二：Electron 桌面应用

```bash
cd electron
npm install
npm start
```

详细说明见 [docs/ELECTRON.md](docs/ELECTRON.md)
